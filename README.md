# ⛽ Petrol Cost Dashboard (Streamlit App)

A modern and interactive **Petrol Cost Calculator** built using **Streamlit** that helps users estimate their fuel expenses for both **daily commuting** and **long trips**.

## 🚀 Features

* 📅 **Monthly Petrol Cost Calculator**

  * Estimate fuel expenses based on daily travel, working days, and mileage range

* 🛣️ **Trip Cost Calculator**

  * Calculate total petrol cost for round trips with customizable mileage

* 📊 **Interactive Visualizations**

  * Dynamic charts showing cost vs mileage

* 💡 **Smart Insights**

  * Displays minimum, maximum, and average fuel costs

* 📋 **Detailed Breakdown**

  * Expandable table view for all mileage scenarios

* 🎨 **Modern UI**

  * Dark-themed responsive design with clean layout and smooth user experience

## 🛠️ Tech Stack

* **Python**
* **Streamlit**
* **Pandas**

## 📦 Installation

```bash
git clone https://github.com/your-username/petrol-cost-dashboard.git
cd petrol-cost-dashboard
pip install -r requirements.txt
streamlit run app.py
```
